cansatapi.message.jsonGenerator.generate_json(time: str | None = None, gps: Gps | None = None, nine_axis: NineAxis | None = None, bme280: Bme280 | None = None, lps25hb: Lps25Hb | None = None, battery: float | None = None, distance: float | None = None, camera: str | None = None, soil_moisture: float | None = None, message: str | None = None) → str[ソース]

JSONを生成する関数

各引数に値を入れると対応するJSONデータを生成します。

パラメータ:

  • message -- 任意のメッセージ

  • soil_moisture -- 土壌水分量[%]

  • camera -- カメラのヘックスデータ

  • distance -- 距離

  • battery -- バッテリー残量[%]

  • lps25hb -- LPS25HBのデータ

  • bme280 -- BME280のデータ

  • time (str) -- データの送信時時間

  • gps (Gps) -- GPSデータ

  • nine_axis (NineAxis) -- 9軸データ

戻り値:

生成されたJSON文字列

戻り値の型:

str
