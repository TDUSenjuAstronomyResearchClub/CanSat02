動作確認用のやりかた

operation_check配下に入れられているモジュールは動作確認用モジュールです。
各センサーに対応したモジュールを実行することで、センサーが正常に動作しているか確認できます。

動作確認用モジュールの使い方

cansatapiをpipでローカルからインストールした後に、動作確認用モジュールを呼び出してください。

# CanSatAPIをインストールする
pip install .

終了時はctrl + cで割り込んでもらえれば停止するようにしています。
